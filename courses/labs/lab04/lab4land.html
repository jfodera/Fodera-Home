Lab4: XML

The purpose of this page is to get familiar with the XML Microformats atom and RSS (really simple syndication). For this lab, I curated two files, one RSS and one Atom. Each of these files have 5 identical entries/items from around the web, the only difference is the syntax. If you view these files with an RSS reader equipped, you should be able to navigate to the entries/items! To navigate to these files, click on the images to the left and right!

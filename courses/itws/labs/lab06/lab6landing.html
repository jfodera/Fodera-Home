Lab6: jQuery

In this lab, we were given starter JavaScript, html, and css files for our solution. We then had to create various functions (specified by the instructions) in which we used newly learned jQuery methods to primarily manipulate elements of the DOM based on user events. There was a heavy emphasis of taking advantage of new methods for each funtion, seemingly in an attempt to give us some experience with a wide range of things discussed in lecture. Overall a fun lab in which I took advantage of jQuery for the first time.
